# Landing Kit 🌐

> Template profissional de landing page multidioma com SEO e captura de leads.

## Stack
Next.js 15 • i18n (next-intl) • TypeScript • Tailwind CSS • Schema.org

## Funcionalidades
- Suporte a 3 idiomas (PT, EN, ES)
- SEO on-page com Schema.org
- Formulario inteligente com validacao
- Analytics e eventos de conversao
- Preparado para A/B testing
- Performance otimizada (Core Web Vitals)

## Metrica
Taxa de conversao 35% maior com formulario inteligente e A/B testing.
